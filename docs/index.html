zip-coords-us

Small, fast ZIP → coordinates lookup library for U.S. 5-digit ZIP codes.

Docs Interactive Demo README

Quick Start

Install and build as described in the project README. The interactive demo will try to load the local UMD build (dist/zip-coords-us.min.js) and fall back to the CDN.

API

The primary API is convert(zip) which returns { latitude, longitude } or null.

convert('10001') → { latitude: 40.7506, longitude: -73.9970 }

Publish

  1. Push this repo to GitHub.
  2. Repository → Settings → Pages → Source: select branch and docs/ folder.
  3. Visit the provided Pages URL.

Built for fast O(1) ZIP → coords lookups. Demo uses UMD global ZipCoordsUS.
