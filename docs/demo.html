Overview

A super-efficient, lazy-loaded map for converting 5-digit US ZIP codes to their centroid latitude/longitude. O(1) lookups after first load.

v1 • ZCTA-based
Lookup
convert(zip)
Return
{ latitude: number, longitude: number } | null

API

Primary function:

// convert accepts string or number (padded to 5 digits)
const coords = ZipCoordsUS.convert('10001'); // { latitude, longitude } or null
Notes:
  • ZIP normalization pads numeric input to 5 digits (e.g. 601 → "00601").
  • Coordinates are ZCTA centroids from U.S. Census Bureau data.

Usage Examples

Node / CommonJS 
const { convert } = require('zip-coords-us');
console.log(convert('10001'))
Browser / CDN (UMD) 
<script src="dist/zip-coords-us.min.js"></script>
<script>ZipCoordsUS.convert('90210')</script>

Build & Publish

  1. Install deps: npm install
  2. Build UMD: npx rollup -c (produces dist/zip-coords-us.min.js)
  3. Publish docs to GitHub Pages: push repo, Settings → Pages → select docs/ folder or use this single-file site in docs/.

The included demo tries to load the local dist file first and falls back to the public CDN.
